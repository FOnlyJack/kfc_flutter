## 功能介绍

- 主要的实现的首页和商城页面多布局、首页布局横向滑动、外送点餐页面布局嵌套滑动效果、帧动画等。
- 首页接口整体数据通过截图和EasyMock弄的,属实蛋疼这操作。

## 计划

- 实现物品加入购物车时贝塞尔曲线动画效果。
